# gotuj-wiadomie
Gotuj świadomie — inteligentny przewodnik kulinarny, który uczy rozumieć procesy w kuchni, a nie tylko podaje przepisy. 96 kart wiedzy o pieczeniu, smażeniu, sosach, deserach i konserwowaniu, interaktywna diagnostyka problemów, przewodnik decyzji i moduł eksperymentów (chleb, stek, biszkopt). Sprawdzone, oparte na chemii kuchni odpowiedzi
